# Building Bayard

## Requirements

The following products are required to build Bayrad:

- Rust >= 1.39.0
- make >= 3.81
- protoc >= 3.9.2

## Build

Build Bayard with the following command:

```text
$ make build
```

When the build is successful, the binary file is output to the following directory:

```text
$ ls ./bin
```
